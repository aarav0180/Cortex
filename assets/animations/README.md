# Agent Avatar Animation

Place your Rive animation file (`agent_avatar.riv`) in this directory.

The animation should have the following state machine inputs (triggers):
- `idle` - Breathing, blinking animation
- `listening` - Bounce animation
- `thinking` - Pulse animation
- `responding` - Mouth movement animation
- `waiting` - Subtle attention animation
- `error` - Confused tilt animation
- `success` - Happy pop animation

The animation should be lightweight (<500KB) for optimal performance.
